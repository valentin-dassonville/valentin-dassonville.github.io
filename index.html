GrowIT.io

Projet de fin du semestre 4

Mars-Avril 2023

Ce dernier projet de 2ème année était de refaire, avec mon binôme, une partie du jeu Agar.io. N'importe qui, à l'aide de son navigateur, peut se connecter au serveur et rejoindre la partie en cours. Le but est de manger, soit de la nourriture, soit des joueurs plus petits que soi afin de devenir de plus en plus gros et d'occuper les meilleures places du classement.

  • Compétences techniques : HTML/CSS, TypeScript, Serveur nodeJs, SocketIO (websocket), Jest, Architecture client-serveur
  • Savoir-être : Esprit d'équipe, efficacité, autonomie, débrouillardise

API REST Pizzeria

Projet du semestre 4

Janvier-Février 2023

Ce projet, que nous avons réalisé en binôme, consiste à créer une API REST afin de gérer sa pizzeria. Cela nous permet via des requêtes http de récupérer, modifier, ajouter des données, stockées sur une base de données postgresql, sur les ingrédients, les pizzas ou encore les commandes.

  • Compétences techniques : API REST, Java, JavaEE, SQL, Design Pattern DAO
  • Savoir-être : Esprit d'équipe, autonomie

Classification k-nn

Projet du semestre 3

Octobre 2022 - Janvier 2023

En équipe de 4, nous avons réalisé une application complexe permettant de : charger un modèle de données depuis un fichier CSV, visualiser un modèle de données graphiquement, choisir les attributs représentant les axes du graphique, classifier les données avec la méthode k-nn sur un attribut. Il est également possible d'ajouter de nouveaux points à classer ou encore définir un critère de sélection sur nos données.

  • Compétences techniques : Java, JavaFX, Algorithme k-nn, MVC
  • Savoir-être : Esprit d'équipe, Polyvalence

Bot Discord

Projet personnel

Juillet-Août 2022

Étant un joueur de League of Legends, j'ai voulu réaliser un bot Discord permettant d'afficher graphiquement les statistiques d'un joueur grâce à une commande. Pour réaliser ce que je voulais faire, j'ai fait des recherches. Il me fallait une bibliothèque afin que mon bot interagisse avec Discord et un accès à l'API de League of Legends.
J'ai donc choisi la bibliothèque JDA car elle est réalisée en Java (c'est le seul langage que je connaisse à ce moment) et j'ai fait une demande pour une clé API à Riot Games (éditeur du jeu) via l'espace développeur qu'ils ont mis en place.
Une fois tout ça mis en place, j'ai pu coder mon programme et réaliser ce que j'avais envie de faire.

  • Compétences techniques : Java, Utilisation d'une API, bibliothèque JDA
  • Savoir-être : Autonomie, Patience, Persévérance, Débrouillardise

Emulateur de processeur

Projet du semestre 5

Septembre 2023 - Janvier 2024

Ce second projet de semestre avait pour but de nous faire faire quelque de nouveau en apprenant un nouveau langage (Rust).
J'ai créé un émulateur d'un processeur comme l'on pouvait avoir dans les débuts, le but étant de pouvoir exécuter du code assembleur. J'ai pu comprendre comment fonctionne un processeur très basique avec les registres et la mémoire. A la fin, il fallait émuler plusieurs processeurs afin de faire une guerre de territoire en capturant le plus de processeur grâce à un programme écrit en assembleur.

  • Compétences techniques : Rust, Assembleur
  • Savoir-être : Autonomie, Autodidacte, Débrouillardise, Résolution de problèmes
