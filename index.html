GrowIT.io

Projet de fin du semestre 4

Ce dernier projet de 2ème année était de refaire, avec mon binôme, une partie du jeu Agar.io. N'importe qui à l'aide de son navigateur peut se connecter au serveur et rejoindre la partie en cours. Le but est de manger, soit de la nourriture, soit des joueurs plus petit que soi afin de devenir de plus en plus gros et occupé les meilleures places du classement.

  • Compétences techniques : HTML/CSS, TypeScript, Serveur nodeJs, SocketIO (websocket), Jest, Architecture client-serveur
  • Savoir-être : Esprit d'équipe, efficacité, autonomie, débrouillardise

API REST Pizzeria

Projet du semestre 4

Ce projet consiste à créer une API REST afin de gérer sa pizzeria. Cela nous permet via des requêtes http de récupérer, modifier, ajouter des données, stockées sur une base de données postgresql, sur les ingrédients, les pizzas ou encore les commandes.

  • Compétences techniques : API REST, Java, JavaEE, SQL, Design Pattern DAO
  • Savoir-être : Esprit d'équipe, autonomie

Caisse

Projet personnel

Projet réalisé à la demande d'un ami restaurateur pendant la période du covid. Il m'a demandé de rajouter à son logiciel unicenta une fonctionnalité : un menu lui permettant d'exporter ses commandes d'un fichier CSV afin de simplifier ses livraisons. Ce projet m'a permis de me plonger dans du code que je ne connaissais pas, de le comprendre et d'y ajouter une nouvelle fonctionnalité.

  • Compétences techniques : Java, SQL, Java Swing/AWT
  • Savoir-être : Autonomie, Débrouillardise

Minecraft

Projet personnel

2018-2021

C'est grâce à Minecraft que je fais des études dans l'informatique aujourd'hui. C'est en voyant les possibilités de créations qu'offrait le jeu que je me suis mis à apprendre Java afin de créer mes propres choses. Un peu plus tard j'ai réussi à rentrer bénévolement en tant que développeur sur un serveur. Cela m'a permis de beaucoup pratiquer et de m'améliorer dans le développement, mais également d'apprendre pleins d'autres choses annexes comme le déploiement d'un serveur, scripts pour faire des backups automatiquement, travailler avec des personnes qui n'y connaissent rien en informatique. Malheureusement, il ne me reste plus beaucoup de traces de ce que j'ai pu réaliser.

  • Compétences techniques : Java, Debian, Bash
  • Savoir-être : Autonomie, Initiative, Esprit d'équipe, Polyvalence, Débrouillardise
